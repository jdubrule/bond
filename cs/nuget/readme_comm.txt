About
=====

Bond Comm is an open source, cross-platform, cross-language framework for
cross-process and cross-server messaging using Bond schematized data.

It is published on GitHub at https://github.com/Microsoft/bond/

The C# documentation is available at
http://microsoft.github.io/bond/manual/bond_cs.html

The C# examples are at
https://github.com/Microsoft/bond/tree/master/examples/cs/comm
